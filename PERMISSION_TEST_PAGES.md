# 权限测试页面使用说明

## 📋 测试页面列表

已创建4个权限测试页面，用于验证RBAC权限系统：

### 1. 登录用户测试页 (`/test/login-user`)

**访问权限**：只要登录就能访问

**权限要求**：`ACCESS_MUNE.USER`

**可见角色**：
- ✅ 所有登录用户（普通用户、教练员、超级管理员）

**功能**：
- 显示当前用户信息
- 显示RBAC角色和权限
- 测试登录用户权限

---

### 2. 普通用户测试页 (`/test/normal-user`)

**访问权限**：普通用户可见，教练员也可见

**权限要求**：`ACCESS_MUNE.USER`

**可见角色**：
- ✅ 普通用户（User）
- ✅ 教练员（Coach）- 因为教练员包含普通用户的所有权限
- ✅ 超级管理员（SuperAdmin）- 因为超级管理员拥有所有权限

**功能**：
- 显示用户信息和权限
- 测试普通用户权限
- 编辑个人资料
- 查看教练资料

---

### 3. 教练用户测试页 (`/test/coach`)

**访问权限**：只有教练员可以访问

**权限要求**：`ACCESS_MUNE.COACH`

**可见角色**：
- ✅ 教练员（Coach）
- ✅ 超级管理员（SuperAdmin）- 因为超级管理员拥有所有权限

**功能**：
- 显示教练员信息和权限
- 测试教练员权限
- 查看绑定用户列表
- 管理绑定用户

---

### 4. 超级管理员测试页 (`/test/super-admin`)

**访问权限**：只有超级管理员可以访问

**权限要求**：`ACCESS_MUNE.SUPER_ADMIN`

**可见角色**：
- ✅ 超级管理员（SuperAdmin）

**功能**：
- 显示超级管理员信息和权限
- 测试所有权限
- 用户管理
- 角色管理
- 系统管理

---

## 🎯 权限继承关系

### 权限层级

```
SuperAdmin (超级管理员)
  └─ 拥有所有权限 ✅
      ├─ Coach 权限 ✅
      └─ User 权限 ✅

Coach (教练员)
  └─ 包含 User 权限 ✅
      └─ User 权限 ✅

User (普通用户)
  └─ 基础权限 ✅
```

### 页面访问权限

| 页面 | 普通用户 | 教练员 | 超级管理员 |
|-----|---------|--------|-----------|
| 登录用户测试页 | ✅ | ✅ | ✅ |
| 普通用户测试页 | ✅ | ✅ | ✅ |
| 教练用户测试页 | ❌ | ✅ | ✅ |
| 超级管理员测试页 | ❌ | ❌ | ✅ |

## 🚀 使用方法

### 1. 访问测试页面

**方式1：通过路由访问**
```
http://localhost:8080/test/login-user
http://localhost:8080/test/normal-user
http://localhost:8080/test/coach
http://localhost:8080/test/super-admin
```

**方式2：通过菜单访问**
- 登录后，在菜单中找到对应的测试页面
- 根据你的角色，会显示相应的页面

### 2. 测试权限

1. **使用不同角色登录**：
   - 普通用户账号
   - 教练员账号
   - 超级管理员账号

2. **访问各个测试页面**：
   - 验证哪些页面可以访问
   - 验证哪些页面不能访问

3. **检查权限显示**：
   - 查看页面上的角色和权限信息
   - 验证权限是否正确显示

## 📊 权限检查逻辑

### 权限检查流程

```
用户访问页面
    ↓
检查是否需要登录
    ↓
加载用户信息和 RBAC 权限/角色
    ↓
权限检查（按优先级）：
  1. SuperAdmin → 直接放行 ✅
  2. 检查需要的权限类型：
     - USER → 所有登录用户都可以访问 ✅
     - COACH → 教练员或超级管理员 ✅
     - SUPER_ADMIN → 只有超级管理员 ✅
    ↓
允许/拒绝访问
```

### 权限检查代码

```typescript
// checkAcess.ts
if (needAcess === ACCESS_MUNE.USER) {
    // 所有登录用户都可以访问
    return true;
}

if (needAcess === ACCESS_MUNE.COACH) {
    // 教练员或超级管理员
    return roleKeys.includes('coach') || isSuperAdmin;
}

if (needAcess === ACCESS_MUNE.SUPER_ADMIN) {
    // 只有超级管理员
    return isSuperAdmin;
}
```

## ⚠️ 注意事项

1. **权限继承**：
   - 超级管理员可以访问所有页面
   - 教练员可以访问普通用户页面（因为包含普通用户权限）
   - 普通用户只能访问自己的页面

2. **权限加载**：
   - 页面加载时会自动加载RBAC权限和角色
   - 如果权限未加载，页面会先加载权限再显示内容

3. **路由守卫**：
   - 未登录用户会被重定向到登录页
   - 无权限用户会被重定向到无权限页面
